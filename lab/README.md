# Lab

This directory is for curated technical experiments that can be reproduced and discussed publicly.

Allowed content:

- minimal adapter fixtures
- XML structure probes
- deterministic compiler experiments
- documented results with no sensitive data

Excluded content:

- competition strategy
- presentation drafts
- raw prompt experiments
- personal TODOs or uncurated notes
- credentials, tokens, account details, or personal identifiers

Experiments that become stable behavior must move into packages, tests, an ADR, or the Tableau reverse-engineering documentation.
